# Differential miRNA Expression Analysis in Nasopharyngeal Carcinoma (GSE70970)

## Overview

This project investigates differentially expressed miRNAs in Nasopharyngeal Carcinoma (NPC) using the GEO dataset GSE70970.

## Dataset

- GEO accession: GSE70970
- Tumor samples: 246
- Normal samples: 17

## Analysis Workflow

1. Data download using GEOquery
2. Data preprocessing
3. Principal Component Analysis (PCA)
4. Differential expression analysis using limma
5. Volcano plot visualization
6. Heatmap generation
7. Identification of significant miRNAs

## Key Findings

- 107 significantly dysregulated miRNAs identified
- Several EBV-encoded miRNAs were highly upregulated
- Host miRNAs such as hsa-miR-145 and hsa-miR-375 were significantly downregulated

## Tools

- R
- GEOquery
- limma
- ggplot2
